O propósito

O aplicativo foi projetado para coletar coordenadas GPS correlacionadas com informações sobre as torres celulares. Ele foi projetado para ajudar a mapear a extensão da cobertura da rede celular.

Como começar

Para começar selecione a opção Iniciar (ícone "play") a partir do menu no tipo da tela. As medições serão coletadas somente quando você está no alcance das redes celulares e o GPS te fornece posição com uma precisão de pelo menos 40 metros.

Próximos passos

Os dados coletados podem ser enviados para o banco de dados do OpenCellID ou exportados para um arquivo. Antes do primeiro envio você deve se registrar gratuitamente para conseguir sua chave de API e inserí-la nas Preferências do aplicativo. As medições serão apagadas do seu dispositivo após o envio com sucesso.
