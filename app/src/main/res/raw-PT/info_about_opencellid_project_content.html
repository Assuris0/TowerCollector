OpenCellID é um projeto de código aberto, visando a criação de um banco de dados mundial de correlação entre CellID e posições GPS. Os dados coletados estão disponíveis sob a licença "Creative Common Attribution-Share Alike 3.0 Unported".
O Tower Collector te dá a oportunidade de contribuir com o projeto OpenCellID ao fazer o upload das torres celulares GSM da sua região. Mais informações.
