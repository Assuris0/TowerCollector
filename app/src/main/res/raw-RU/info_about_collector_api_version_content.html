Вы можете решить какой из методов будет использоваться для получения информации о вышки во время сбора измерений.
Очень рекомендуется ипользовать последний доступный для сбора большего числа данных. Секция недоступна на устройствах под управлением Android 4.1 и менее.

  • Автоопределение - приложение автоматически определяет какую версию API использовать и выбирает лучшую. Также выбирается безопасная опция, если Сборщик запускается когда нет сигнала.
  • Android 4.2 API - использует методы доступные на Android 4.2 и новее. Эти методы получают наиболее полную информацию о ячейке сотовой сети. Однако, некоторые производители неверно используют новое API и его невозможно использовать на таких устройствах. На Android 5.1 и новее ячейки сотовой связи с нескольких SIM карт могут быть возвращены.
  • Android 1.0 API - Используются методы, доступные с первой версии Android. Эти методы могут возвращать недостоверные данные для сетей LTE и UMTS. В зависимости от устройства, они могут требовать постоянно включённого экрана, чтобы собирать информацию корректно.
