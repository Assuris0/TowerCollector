OpenCellID - это проект с открытым исходным кодом, стремящийся создать всемирную базу данных GPS-сот с привязкой их к координатам GPS. Собранные данные доступны по лицензии "Creative Common Attribution-Share Alike 3.0 Unported".
Tower Collector позволяет внести вклад в проект OpenCellID путём загрузки собранных данных вашей местности. Дополнительная информация.
