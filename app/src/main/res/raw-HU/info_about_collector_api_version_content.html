Kiválaszthatod a cellainformáció gyűjtésének módját.
Ajánlott a legfrissebbet választanod, hogy megbízható adatokat kapj. Android 4.1 és korábbi készülékeknél nem érhető el.

  • Automatikus választás - Az alkalmazás meghatározza az API verzióját és automatikusan kiválasztja a legjobbat. Biztonságos beállítást használ, ha a gyűjtést olyan helyen indítod, ahol nincs térerő.
  • Android 4.2 API - Felhasználja az Android 4.2 és újabb változatokban elérhető legjobb adatlekérést. Egyes gyártók sajnos nem készítették el megfelelően, ezért az ő készülékeiken nem használható. Android 5.1-től kezdve több SIM kártyához tartozó cella-adatok is rögzíthetők.
  • Android 1.0 API - A legelső Android változatban megjelent adatlekérést használja. Ez nem ad megbízható adatot LTE és UMTS hálózatokban. A készülék típusáttől függően szükség lehet a képernyő ébren tartására, hogy a cellák adatai frissüljenek.
